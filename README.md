# exotic_option_pricing

# Ценообразование экзотического опциона: Барьерный пут «наихудший-из-двух» с понижением и активацией

Оценка стоимости и анализ рисков экзотического опциона на две европейские нефтяные компании — Shell (SHEL) и TotalEnergies (TTE) — методом Монте-Карло с разложением Холецкого.


Данная структура сочетает три экзотических компонента:

- **Барьер (Down-and-In)** — опцион становится активным только после того, как хотя бы одна акция упадет до 70% от своей начальной цены (событие активации)
- **Корзина** — опцион написан на два коррелированных актива, поэтому корреляция становится ключевым фактором ценообразования
- **Наихудший-из-двух (Worst-of)** — в момент экспирации выплата зависит от наихудшей из двух акций

**Формула выплаты:**
$$
\text{Payoff} = \mathbb{1}_{\text{knock-in}} \times \max\left(K - \min\left(\frac{S_1(T)}{S_1(0)},\ \frac{S_2(T)}{S_2(0)}\right),\ 0\right)
$$

Опционы такого типа часто встраиваются в структурированные продукты с обратной конвертируемостью (reverse convertible) — облигацию с повышенным купоном, где инвестор получает доход за счет продажи этого пут-опциона.

Моделирование Монте-Карло — стандартный численный метод для оценки стоимости производных инструментов с множеством базовых активов и зависимостью от пути.

## Коррелированные траектории через разложение Холецкого

Поскольку SHEL и TTE коррелированы (ρ ≈ 0.76), мы не можем моделировать их независимо. Используем разложение Холецкого для генерации коррелированных случайных величин:
$$
L = \begin{pmatrix} 1 & 0 \\ \rho & \sqrt{1-\rho^2} \end{pmatrix}, \quad
\varepsilon_1 = Z_1, \quad
\varepsilon_2 = \rho Z_1 + \sqrt{1-\rho^2}\, Z_2
$$
где $Z_1, Z_2$ — независимые стандартные нормальные величины.

## Динамика GBM

Каждая акция следует геометрическому броуновскому движению в риск-нейтральной мере:
$$
S_i(t+dt) = S_i(t) \cdot \exp\left[\left(r - q_i - \tfrac{1}{2}\sigma_i^2\right)dt + \sigma_i\sqrt{dt}\,\varepsilon_i\right]
$$

## Греки

Все греки вычисляются методом центральных конечных разностей с использованием общих случайных чисел (один и тот же seed для всех переоценок для снижения шума Монте-Карло):

| Грек | Что меняем | Формула |
|------|------------|---------|
| Дельта (Delta) | Спот ±1% | $(V^+ - V^-) / (2\Delta S)$ |
| Гамма (Gamma) | Спот ±1% | $(V^+ - 2V_0 + V^-) / (\Delta S)^2$ |
| Вега (Vega) | Волатильность ±1% | $(V^+ - V^-) / (2\Delta\sigma)$ |
| Кросс-гамма (Cross-Gamma) | Оба спота ±1% | $(V_{++} - V_{+-} - V_{-+} + V_{--}) / (4\Delta S_1 \Delta S_2)$ |
| Чувствительность к корреляции | ρ ±1 б.п. | $(V^+ - V^-) / (2\Delta\rho)$ |

**Важная деталь реализации:** при изменении цены спот для расчета дельты и гаммы барьер и начальная цена для нормировки доходности должны оставаться фиксированными. Иначе изменение масштабирует всё пропорционально, и греки обнуляются.

## Результаты

| Показатель | Значение |
|------------|----------|
| Цена опциона | 4.78% от номинала |
| Вероятность активации (knock‑in) | 16.1% |
| Дельта (SHEL / TTE) | –0.16 / –0.23 |
| Гамма (SHEL / TTE) | +2.11 / +2.13 |
| Вега (SHEL / TTE) | +0.30 / +0.40 |
| Кросс-гамма | –0.69 |
| Чувствительность к корреляции | –0.029 |


## Ограничения модели

- **Постоянная волатильность** — отсутствие улыбки/скоса; поверхность подразумеваемой волатильности может изменить реальную цену на 20–30%
- **Отсутствие скачков** — GBM не захватывает ночные гэпы, характерные для нефтяных акций
- **Постоянная корреляция** — в реальности ρ стохастична и имеет тенденцию расти во время кризисов
- **Историческая волатильность как прокси** — производственные системы используют подразумеваемую волатильность
